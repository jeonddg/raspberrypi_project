01 Overview

프로젝트 개요

목표

현재 습도만 기준으로 켜지고 꺼지는 단순 가습기보다 똑똑한 제어 시스템을 구현합니다. 실내 온습도, 외부 날씨, 최근 습도 변화량, 물 수위 상태를 함께 고려해 가습 필요성을 판단합니다.

핵심 아이디어

AI 모델이 10분 후 실내 습도를 예측하고, 예측 결과가 건조 범위로 내려갈 가능성이 높으면 가습기를 미리 동작시킵니다. 물 부족이나 과습 상황에서는 안전하게 가습을 중단합니다.

03 Features

주요 기능

실내 온습도 측정

DHT 센서로 현재 실내 온도와 습도를 측정하고 LCD와 대시보드에 표시합니다.

외부 날씨 연동

OpenWeatherMap API로 외부 온도, 습도, 날씨 상태를 가져와 예측 보조 데이터로 사용합니다.

물 수위 감지

Water Detection Sensor와 PCF8591 ADC를 사용해 물 부족 상태를 감지합니다.

자동 제어

릴레이 모듈로 초음파 가습 모듈 전원을 제어하고, 경고 상황은 LED와 부저로 알립니다.

04 AI Model

AI 습도 예측 모델

입력 데이터 설명
indoor_temp 현재 실내 온도
indoor_humidity 현재 실내 습도
outdoor_temp / outdoor_humidity 외부 날씨 API 데이터
humidity_change_5min 최근 5분간 습도 변화량
humidifier_state / water_level 가습기 동작 상태와 물 수위
